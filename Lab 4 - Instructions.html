Course Lab 4

Lab Information and Requirements

Lab Directions:

  • You will be using the project from Lab 3.
  • Add a meta viewport tag to all pages on the site.
    • https://www.w3schools.com/tags/tag_meta.asp
  • Using an external stylesheet style, the following items:
    • Body: background- color and set a global font-family
    • Img: set max-width: 100%.
    • Wrap div id called container around all the tags in the body. 
    • In the container add the following styles:
      • Set the container width to 75%
      • Center the container using margin 0 auto
      • Give the container a background color.
      • Add 20px of padding
      • Add 2px border solid with a color of your choosing.
    • Header h1: font color, font-size: 60px, and center the h1 in the header.
    • Style the nav tag to look like a navbar.
    • Main: add 10px of padding and margin.
    • Main h2: change the color, size and center.
    • Add the following styles to the image in the main tag.
      • Center the image
      • Add 20px of border-radius
      • Change image width to 75%.
    • Add the following styles to the footer
      • Background color
      • 40px top and bottom 0 left and right
      • Center the text
      • Change the font color

Lab Requirements:

  • Once the lab is complete:
    • Upload the site to your site to GitHub (Be sure you have watched the video about get GitHub).
    • Validate your code
    • Create a word document with a link to your site. Include screenshots of your validated code in the document. There should be no errors.
    • Upload the word doc to the dropbox on eLearn.
